# 看信号做决策 — 信号评估标准原文

## 根节点：看信号，做决策

## 一、盘面结构信号

### 1. 共振
- **指数共振**（沪深300、上证、创业板）
  - 同步上涨视为及格
  - 均涨2%视为突破
- **权重与小票共振**（沪深300、中证2000、分时黄白线）
  - 黄白线剪刀差1.5%以内视为正常

### 2. 权重情况
- **沪深300成分股**（上涨超5%与下跌超5%的比例）
  - 无下跌5%、比例超过3，视为良好
- **创业板6大权重**：宁德、阳光、东财、迈瑞、汇川、中际
  - 综合上涨视为良好

### 3. 题材情况
- **主线板块**
  - 有无主线、主线炒作处于什么阶段
- **连板指数**
  - 是否上涨，是否创新高

### 4. 涨跌情况
- **涨跌家数、涨跌停数量**
  - 60家涨停视为及格
  - 80家涨停视为突破
